Горячие клавиши.
    Для красоты.
        1 Полностью задокумментировать несолько строк: Ctrl + /
        2 Выпадающие списки: Ctrl + .
    Для оформления.
        1
    Для кода.
        1 psmv + Tab - public static void main(String[] args)
        2 sout -  System.out.println
    Для работы с текстом
        1 Shift+Ctrl+V - Просмотр буфера данных
        2 Shift+Alt+Клик - Поставить множественные курсоры для ввода текста
        3 Ctrl+O - удалит все неиспользуемые импорты
        4 Ctrl+Alt+t - обернуть в.
        5 Ctrl+Alt+L - для форматирования кода.


Правила.
    1 Убираем пробелы.

    2 Комментрации
        Бывают трех видов Однострочные, Многострочные и Документационные.
            -Однострочные - //
            -Многострочные - /* */
            -Документационные - /**

    3 Логические операторы
        if
        else
